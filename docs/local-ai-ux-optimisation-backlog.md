# Local AI UX Optimisation Backlog

Future ideas intentionally left out of the proof of concept:

- FullStory integration
- FullStory MCP integration
- Real AI suggestion generation
- Heatmaps
- Funnels
- Session replay
- Dead click detection
- Rage click detection
- Journey scoring
- Recommendation confidence scores
- Approve/reject learning loop
- Automatic implementation via Codex
- Import analytics into another tool
- Visual dashboard charts
- Export/import controls for suggestion state
- Per-component experiment configuration
- Multi-session comparison
- Privacy and consent controls

The current prototype is deliberately local and small: capture behaviour, generate rule-based suggestions, approve or reject them, and apply only tiny safe UI changes.
